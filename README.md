# vue-weather-widget

Weather Widget

### Technology stack

- Vue3 (Composition API)
- TypeScript
- Vite

### Features

- Requests your current position.
- Can show the weather in three cities.
- You can change the order of the cities.
- Stores in the browser's local storage.

## Usage

1. Add the following code snippet in your page (html file)
   ```html
   <script
     type="module"
     src="https://cdn.jsdelivr.net/gh/MyWay778/vue-weather-widget/dist/weather-widget.min.js"></script>
   <weather-widget></weather-widget>
   ```

## Development setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Create dist

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
